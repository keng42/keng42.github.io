在 Android Studio 同步项目到 Github

Jul 20, 2015 in 加工

步骤

  1. 电脑本地安装 Git
  2. 在 Android Studio 中配置 git 路径
  3. 在 Android Studio 中设置 Github 登陆信息
  4. 将项目初始化 git 项目
  5. 安装忽略文件的插件
  6. 发送到 Github
  • 直接安装了 Windows 版的 Github 客户端，git 路径为 %user%\AppData\Local\GitHub\PortableGit_<...>\bin\git.exe
  • File -> Setting -> Version Control -> Git
  • VCS -> Enable Control Integration -> Select “Git”
  • VCS -> Import into Version Control -> Share Project on GitHub
