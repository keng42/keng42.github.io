几个单位

px pixel 像素
dpi dots per inch 每英寸点数
ppi pixels per inch 每英寸像素
dip density independent pixel 密度无关像素
dp == dip
这么说，px 是真实的物理单位，对安卓手机来讲标准量化的 dpi 有6种（当然还有其他的），分别是 ldpi, mdpi, hdpi, xhdpi, xxhdpi, xxhdpi, 以 mdpi == 160 为标准，其余的按倍数缩放，分别是 0.75, 1, 1.5, 2, 3, 4, 这些倍数就是 logical density 逻辑密度，这样就可以根据逻辑密度来进行 px 和 dp 的换算—— px = dp * density。
Android 里可以通过这个方法来获取相关的信息，包括 density, xdpi, ydpi, widthPixels, heightPixels…

总的来说，字体大小用 sp, 其余的大小用 dp

dp 和 px 相互转化

监听横竖屏切换

  1. 在 AndroidManifest.xml 中的 Activity 配置中添加 android:configChanges="orientation|screenSize"

    从 Andoid 3.2(API level 13) 开始，横竖屏切换的时候会改变屏幕尺寸，所以为了避免在切换屏幕时重启 Runtime，除了 orientation 之外还要添加 screenSize。另外，如果你的 targetSdkVersion <= 12，那么，无论什么版本的安卓都不会在在切换屏幕时重启 Runtime。
