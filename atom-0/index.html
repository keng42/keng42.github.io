Atom 使用记录

Jul 15, 2015 in 工具

Package

手动安装包

从 Github 下载 zip 文件，解压到 %user%\.atom\packages 下，重启 Atom

格式化代码 atom-beautify

包括 HTML CSS JS Java Markdown
手动安装后需要在项目目录安装缺失的东西

快捷键

自定义快捷键

为了和 Android Studio 保持一致

Action Key
Show/hide Tree-View ctrl+k,ctrl+b
Markdown preview ctrl+shift+m
Settings ctrl+ ,
Add/remove a selection Alt + Shift + Mouse Click
Select/unselect the next occurrence Alt + J / Shift + Alt + J
Select all occurrences Shift + Ctrl + Alt + J
Remove all selections Esc

现在的缺点

  • 体积太大，解压后211MB
  • 时不时会卡死，CPU 彪到 100%
  • 首次启动太久，现在目测需要3秒
  • 1.0.2 使用自带的包安装工具安装会出错，目测是墙的问题

优点

  • 开源
  • 自带 Github Markdown 预览
  • 超方便的快速补完
  • 中文支持比较好
  • 可自定义程度很高

其他问题记录

  • 问题：默认保存文件时，会弹出警告框 你不能使用该程序打开此文件...

    原因：默认是 config 项目，路径是 atom://config
    解决：添加一个新项目并移除 config 项目
