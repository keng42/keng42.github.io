Android Studio 插件

Jul 20, 2015 in 工具

常用插件

安装插件：File -> Settings -> Plugins -> Browser Repositories
插件设置：File -> Settings -> Other Settings

Statistic

显示项目的统计数值。包括源代码行数、注释和空白行数，各种文件的大小。

Excluded directories 排除项目自动生成的文件和导入的类库文件 /app/build/ /app/libs

QAPlug

使用 CheckStyle, FindBugs, PMD 来提高你的安卓代码质量
使用介绍： 如何提高你的代码质量 How to improve quality and syntax of your Android code
使用记录： 使用 Android Studio 的 QAPlug 插件对代码进行优化的记录

ButterKnife Zelezny

根据 layout 文件自动生成 ButterKnife 视图注入代码，告别没完没了的 findViewById()

记得添加依赖 compile 'com.jakewharton:butterknife:7.0.1'

GsonFormat

根据 JSON 字符串生成实体类

Android strings.xml Tools

字符串排序和多语言管理工具

推荐几个非常有用的工具
