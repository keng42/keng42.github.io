Jul 04, 2015

搭建博客

  1. 安装 Git 和 Node.js
  2. 新建文件夹 Github\hexo

  3. 使用 Git Bash

  4. 使用 hexo s 并打开 localhost:4000 测试查看效果

使用主题 tranquilpeak

  1. 在hexo themes中选择主题并下载
  2. 解压文件和子文件夹到 GitHub\hexo\themes\tranquilpeak
  3. 修改 _config.yml 内的 theme 设定

使用 Disqus 添加评论功能

  1. 在 Disqus 中注册并获取 shortname
  2. 修改 tranquilpeak\_config.yml 内的 disqus_shortname: <shortname>
